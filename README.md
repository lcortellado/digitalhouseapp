### App Mobile


1. Ir al directorio del proyecto<br/>
  `cd digitalHouseapp`
2. Instalar dependencias<br/>
  `yarn install` o en su defecto `npm install`
3. Ejecutar el comando<br/>
  `yarn ios` o  `npm android` para elegir el dispositivo en que se quiera ejecutar.
4.
